# Android AI 开发助手

基于大语言模型的 Android 项目辅助工具，集成代码生成、自动应用、版本管理、SSH 终端与远程部署。

## ✨ 主要特性

- 🤖 多 API 支持（智谱 / OpenAI / DeepSeek），深度思考过程可见  
- 📁 加载项目后自动识别包名，AI 可读取全部文件作为上下文  
- 📝 AI 代码块自动解析路径注释（`// file: 相对路径`）并覆盖写入项目  
- 💾 每次应用前后自动备份，支持版本历史与一键回退  
- 🖥️ 内置异步 SSH 终端，命令执行不阻塞界面  
- ☁️ 一键部署 `backend/` 文件至远程服务器（按包名隔离目录）  
- 🎨 多款深色/浅色主题自由切换  

## 🚀 快速开始

```bash
pip install customtkinter paramiko zai openai
python CodeHelper.py
```

## 📖 基本使用

1. **加载项目**：点击“选择项目”打开 Android 根目录。  
2. **AI 对话**：输入需求，按 `Ctrl+Enter` 发送；勾选“自动附加项目文件”可提供完整上下文。  
3. **代码应用**：AI 回复中的代码块若含 `// file: app/src/.../Main.kt` 等路径注释，工具会自动写入并备份版本。  
4. **版本管理**：点击“版本历史”查看所有快照，可回退任意版本。  
5. **SSH 终端**：右侧终端输入命令，自动连接服务器（设置中配置）。  
6. **部署后端**：生成 `backend/` 文件后，点击“部署后端”上传至远程服务器。

## ⚙️ 配置

- **API 管理**：设置 → API管理，可添加/切换/测试多套 API 配置。  
- **服务器配置**：设置 → 服务器配置，填写 SSH 连接信息并保存。  
- **主题切换**：设置 → 主题选择，切换后需重启生效。

## ⚠️ 注意事项

- 请确保 API Key 有效，否则 AI 功能不可用。  
- 版本备份会占用磁盘空间，请定期清理 `.ai_assistant_versions/` 旧目录。  
- SSH 连接超时或失败请检查网络和防火墙设置。
